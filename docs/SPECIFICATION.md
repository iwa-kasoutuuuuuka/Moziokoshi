# MoziOkoshi Pro 仕様書 (v1.3.0)

## 概要
MoziOkoshi Proは、OpenAI Whisper (Faster-Whisper) をベースにした、究極のパフォーマンスと実用性を追求した音声・動画文字起こしアプリケーションです。

## 技術的特徴 (Pro Edition)

### 1. 推論の高速化 (Speculative Decoding)
`large-v3-turbo` モデルの推論を `tiny` モデル（アシスタント）で支援。生成プロセスの一部を並列・投機的に実行することで、認識精度を維持したままスループットを向上させています。

### 2. 計算リソースの最適化 (Numba JIT)
Python の動的なオーバーヘッドを排除するため、以下の処理を Numba により LLVM マシンコードへ JIT コンパイルしています。
- **音声信号処理**: NumPy 配列の正規化（AVX/AVX2/AVX-512 命令の活用）。
- **テキスト処理**: Trie 木アルゴリズムによる高速な単語マッチング（置換・除去）。
- **タイムスタンプ整形**: 浮動小数点演算から文字列生成までのマイクロ最適化。

### 3. ディスク I/O の排除 (In-Memory Processing)
FFmpeg のパイプ機能を利用し、音声データを一時ファイルとして書き出すことなく、直接 NumPy 配列としてメモリへロードします。これにより、SSD/HDD の負荷を軽減し、全体的なレイテンシを最小化しています。

### 4. ビルドの軽量化
PyInstaller のビルドプロセスを独自に最適化。
- **不要モジュールの除外**: OpenCV, Matplotlib, SciPy, Pandas, Scikit-learn 等、10GB以上の不要なライブラリを徹底的に排除。
- **資産の選択的コピー**: Hugging Face の巨大なキャッシュフォルダ（`.cache`）をコピーせず、実行に必要な実体ファイルのみを同梱。
- **最終サイズ**: 15GB から 約6.2GB への削減に成功。

## ファイル構造
```
MoziOkoshi/
├── core/                 # コアロジック (Numba最適化済み)
├── ui/                   # GUI (PySide6 / カスタムQSS)
├── utils/                # ユーティリティ (ロギング等)
├── ffmpeg/               # 実行に必要なFFmpegバイナリ
├── models/               # 必要なモデルデータ (large-v3-turbo, tiny)
├── replacement_dict.txt  # カスタム置換辞書
└── filler_dict.txt       # フィラー除去辞書
```

## パフォーマンス指標
- **処理速度**: large-v3-turbo 使用時、音声実時間の約 1/10 〜 1/5 (NVIDIA RTX 3060 以上環境)
- **メモリ使用量**: 約 3GB 〜 5GB (処理モードに依存)
- **配布形式**: 完全ポータブル版 (実行ファイル + 内部ライブラリ一式)

## セキュリティ
- **100% ローカル処理**: インターネットへの音声データ送信は一切行われません。
- **プライバシー保護**: 入力ファイルや認識結果が外部サーバーに記録されることはありません。
